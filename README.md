# TW-Semiconductor-Analysis-2026
台灣半導體產業 K 線基準化分析

## 專案簡介
本專案為自動化的金融數據視覺化系統。透過擷取台灣半導體產業鏈（包含 IC 設計、晶圓代工與封測、設備與材料）共 15 檔指標股的交易數據，實作等權重基準化，藉此排除千元高價股對產業指數的視覺扭曲，還原市場資金真實的輪動軌跡。

---

##  使用與執行須知

為了確保程式碼能順利執行並產出正確的分析圖表，請詳細閱讀以下須知：

### 1. 系統與環境配置須知
本專案於 Jupyter Notebook 環境下開發。執行前，請確保您的本機環境已安裝 Python 3.11 (或以上) 版本，並透過終端機 (Terminal/CMD) 安裝以下核心依賴套件：
`pip install yfinance pandas mplfinance`

### 2. 程式執行與自定義須知
* **執行方式：** 打開 ipynb 檔，依序執行程式碼區塊即可。
* **資料獲取時間：** 程式預設抓取近 120 天的歷史資料，可自動過濾無效交易日。
* **自定義觀察名單：** 若欲觀察其他產業，可直接至程式碼中的 `stock_categories` 字典修改。**請注意：本腳本專為台股設計，股票代碼後綴必須保留 `.TW`（上市）。**

### 3. 圖表邏輯與判讀須知 (重要)
本系統產出的 K 線圖採用了「量價解耦」的混合維度模型，請依據以下標準進行趨勢判讀：
* **上方 K 線 (Normalized Price)：** 採用等權重基準化。起點強制設為 `100`，數值代表該族群的「平均累積報酬率」。(例：105 代表該族群上漲 5%)。
* **下方成交量 (Average Volume)：** **未經基準化處理**。保留 5 檔股票的「絕對平均交易張數」，用以觀測真實的市場資金流動性。**勿將上方的百分比與下方的絕對張數進行強制因果綁定。**

### 4. API 限制與網路須知
本程式透過 `yfinance` 串接 Yahoo Finance 開源 API 獲取即時數據。執行時必須保持網路連線暢通；若遇 API 伺服器不穩定或台股休市期間，可能會導致最新一日的數據延遲更新。
